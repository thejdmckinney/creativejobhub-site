# Blog Images

This directory contains header images for blog posts.

- `choose-field-service-software-1200.svg` - Software comparison visualization with cards, scores, and evaluation checklist
- `getting-started-1200.svg` - Step-by-step roadmap showing the journey from setup to success
- `choose-field-service-software-1200.jpg` - Legacy JPG version (can be removed)
- `getting-started-1200.jpg` - Legacy JPG version (can be removed)

## Image Specifications

- **Format**: SVG (vector graphics for crisp display at all sizes)
- **Dimensions**: 1200x600px (2:1 aspect ratio)
- **Style**: Dark theme matching site design
- **Colors**: Using site color palette (brand blue, success green, etc.)

## Design Guidelines

Each image should:
- Clearly relate to the blog post title and content
- Use consistent dark theme background
- Include relevant icons or illustrations
- Maintain professional, modern appearance
- Be optimized for web display
