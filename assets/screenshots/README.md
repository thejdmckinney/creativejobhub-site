# App Screenshots Directory

This directory contains screenshots of the Creative Job Hub application interface for the app showcase gallery.

## Required Screenshots

The app-screenshots.html page expects the following screenshot files:

1. `dashboard-overview.jpg` - Main dashboard view with metrics and job status
2. `scheduling-interface.jpg` - Smart scheduling system with AI route optimization
3. `mobile-job-sheet.jpg` - Mobile app interface for job management
4. `customer-portal.jpg` - Customer-facing portal interface
5. `invoicing-system.jpg` - Invoicing and payment processing interface
6. `analytics-reports.jpg` - Business intelligence and reporting dashboard

## Image Requirements

- **Format**: JPG or PNG
- **Dimensions**: Minimum 800x600px, recommended 1200x800px
- **File Size**: Keep under 500KB for fast loading
- **Quality**: High-resolution screenshots showing the actual app interface
- **Content**: Real or realistic mockup data, avoid placeholder text

## Fallback

If screenshot files are missing, the page will automatically fall back to placeholder illustrations from `/assets/illustrations/placeholder-dashboard.svg`.

## Adding New Screenshots

1. Take high-quality screenshots of the actual app
2. Optimize images for web (compress without losing quality)
3. Save with the exact filenames listed above
4. Test the app-screenshots.html page to ensure proper loading
