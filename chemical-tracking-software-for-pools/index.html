Pool Chemical Tracking Software FAQs

What makes the chemical calculator "smart" and CPO-certified?

Our calculator uses algorithms based on CPO (Certified Pool Operator) standards from the Pool & Hot Tub Alliance (PHTA) and APSP. It accounts for pool volume (gallons), current water temperature, and complex chemical interactions (like how pH affects chlorine effectiveness). When you enter test results, it calculates exact dosages needed using industry-proven formulas‚Äînot simple rule-of-thumb guesses. For example, lowering pH from 8.0 to 7.5 in a 20,000-gallon pool requires different acid amounts than a 10,000-gallon pool, and our calculator factors this precisely.

Do I need special water testing equipment to use this software?

No! Use whatever water testing method you currently use‚Äîtest strips, drop test kits (Taylor), or digital testers. The software accepts manual entry of test results. Test the water with your preferred method, then enter the 7 parameter readings into the mobile app. The smart calculator works with readings from any testing equipment. Most pool techs use LaMotte or Taylor test kits (costs $30-80) which are perfectly compatible.

How does the software track chemical inventory on trucks?

You set initial inventory levels for each chemical type per technician truck (e.g., Tech John has 50 lbs chlorine, 10 gallons muriatic acid). When techs log treatments applied ("added 3 lbs chlorine"), the system automatically deducts from that truck's inventory. The dashboard shows real-time inventory levels per truck. You get low-stock alerts when chemicals drop below threshold (e.g., "Tech John has 5 lbs chlorine left‚Äîrestock needed"). Weekly restock reports tell you exactly what to buy and load on each truck.

Can I see historical water chemistry trends for problem pools?

Absolutely! Every water test is saved with date, readings, and treatments applied. View any pool's complete chemical history since you started using the software. The system generates trend charts showing how chlorine, pH, and other parameters changed over weeks and months. This is invaluable for troubleshooting chronic issues‚Äî"This pool's alkalinity has been dropping steadily for 2 months" or "Phosphates spiked in July and stayed high." Use history to identify patterns and solve recurring problems rather than treating blindly each visit.

How does auto-billing work for chemicals used?

You set chemical prices in the system with cost + markup percentage (e.g., chlorine costs you $2.50/lb, you mark up 80% = bill customer $4.50/lb). When techs log treatments applied during service ("added 3 lbs chlorine at Smith pool"), the system calculates cost (3 lbs √ó $4.50 = $13.50) and adds a line item to the Smith's running invoice. At month-end, the recurring invoice includes all regular service visits PLUS itemized chemical charges: "3 lbs chlorine - $13.50, 16 oz muriatic acid - $3.20." Customers see exactly what chemicals were used. You capture revenue automatically.

Does the mobile app work offline at pools without cell service?

Yes! The mobile app works fully offline. Techs can test water, enter results, get treatment recommendations, log chemicals applied, add notes, and take photos‚Äîall without internet. The calculator runs locally on their phone using the pool's stored data (volume, equipment, history). When the tech gets back in cell coverage or connects to WiFi, all data syncs automatically to the cloud. This is essential for pool routes with spotty coverage or when working in areas with poor signal.
