Initial Setup

Organization Settings

Location: Settings → Organization

  1. Enter organization name, timezone, and contact details.
  2. Upload company logo (PNG/JPG, ~200×200px). Add address, phone and email (used on PDFs).
  3. Customize PDF terms and save.

Team Setup

Location: Settings → Team

  1. Invite team members; assign roles (Owner, Manager, Technician, Admin).
  2. Set hourly rates, departments, and employee details; send invitation.

Billing Setup

Location: Settings → Billing

  1. Select plan (Starter / Professional / Enterprise). Add subscription payment method.
  2. Configure Stripe in Settings → Stripe to accept customer payments. New accounts typically have a 7-day free trial.

Client Management

Adding Clients

Location: Clients → Add Client

  1. Fill required fields: name, email (for portal), phone (for SMS), address, notes. Save.
  2. Best practice: always include an email for portal access and a phone for notifications.

Import via CSV

Location: Settings → Import

  1. Download CSV template, populate records, upload and validate, then import valid rows.
name,email,phone,address1,city,state,zip,notes

Client Details

  • View full profile, properties, job history, estimates, invoices and event log.
  • Actions: send email/SMS, create estimate/job, record payments, view portal link.

Property Management

Creating Properties

Location: Properties → Add Property

  1. Select client, enter full address, add notes or access instructions, save.
  2. Properties enable mapping, routing and maintenance history per location.
  3. Upload property photos and documents for easy reference.

Recurring Maintenance

Location: Property Details → Add Recurring Maintenance

  1. Set frequency, start date, service type and default technician. System generates jobs automatically (typically 7 days before due date).
  2. Configure notifications to alert both your team and customers.
  3. View upcoming maintenance schedule on dashboard.

Property Features

  • Track full service history per property
  • View property location on map
  • Store access codes and special instructions
  • Link multiple properties to one client

Lead Management

Lead Statuses

  • New, Contacted, Qualified, Won, Lost

Lead Sources & Tracking

  • Track lead source (Website, Referral, Phone, etc.)
  • Map view with heatmap visualization
  • Service area verification with geolocation
  • Automatic lead capture via AI receptionist

Workflow

  1. Contact leads within 24 hours, qualify, create estimates, follow up, convert to client when won.
  2. Use lead nurturing sequences for automated follow-ups (see Service Plans section).
  3. View lead analytics and conversion rates in Reports.

Job Management

Creating Jobs

Location: Jobs → New Job

  1. Create from an approved estimate or standalone; fill title, description, priority, status, assign technicians, schedule, add line items and save.

Status Flow

  1. Pending → Scheduled → In Progress → Completed → Cancelled

Advanced Job Features

  • Line Items: Add from price book or manually
  • Expenses: Track materials, receipts and costs
  • Photos: Upload with GPS timestamps and AI analysis
  • Documents: Generate warranties and scopes of work
  • Time Tracking: Clock in/out directly on jobs
  • Tasks & Checklists: Break jobs into subtasks
  • Live GPS Tracking: Monitor technician locations in real-time
  • Job Sessions: Track multiple work sessions per job
  • Email & SMS: Communicate directly from job page

Jobs Map View

Location: Jobs → Map View

  • Interactive map with job markers and clustering
  • Filter by status, date range, and technician
  • View real-time technician locations
  • Route planning and optimization tools
  • Click markers for quick actions

Estimates & Smart Estimator

Creating & Sending Estimates

Location: Estimates → New Estimate

  1. Create estimate, add line items (from price book), set deposit if required, save and send.
  2. Track views: see when client opens estimate
  3. Convert accepted estimates to jobs with one click
  4. Send reminders for pending estimates

Smart Estimator (AI-Powered)

Location: Estimates → Smart Estimator

  1. Input job details: Service type, property info, specific requirements
  2. AI Analysis: System reviews your pricing history, market data, and job complexity
  3. Three-Tier Pricing: Receive Good, Better, Best pricing options with detailed breakdowns
  4. Each tier includes:
    • Total price with labor, materials, travel, overhead breakdown
    • Detailed line items with quantities and unit prices
    • When-to-use recommendations
    • Market insights and competitive analysis
  5. One-click convert: Select tier and create full estimate instantly

Estimate Workflow

  • Draft: Being prepared
  • Sent: Delivered to client (tracks opens)
  • Viewed: Client has opened estimate
  • Accepted: Client approved (convert to job)
  • Declined: Client rejected

Integrations

Stripe (Payment Processing)

Location: Integrations → Stripe or Settings → Stripe

  1. Click "Connect Stripe Account"
  2. Complete Stripe OAuth authorization
  3. Stripe automatically processes invoice payments
  4. Standard fees apply (2.9% + $0.30 per transaction)

Twilio (SMS & Voice)

Location: Settings → Messaging

  1. Get Twilio Account SID, Auth Token from twilio.com
  2. Purchase Twilio phone number
  3. Enter credentials in Creative Job Hub
  4. Enable SMS messaging and voice calls
  5. Configure AI receptionist (optional)

QuickBooks Online

  3. Two-way sync:

Document Templates

Creating Templates

Location: Settings → Document Templates

  1. Create custom PDF templates for warranties and scopes of work
  2. Use HTML/CSS for styling
  3. Add dynamic fields with Handlebars syntax: {{`{{client_name}}`}}, {{`{{job_no}}`}}, etc.
  4. Customize header, content, and footer sections
  5. Preview before saving

Available Variables

  • {{`{{client_name}}`}}, {{`{{client_email}}`}}, {{`{{client_phone}}`}}
  • {{`{{property_address}}`}}, {{`{{property_city}}`}}
  • {{`{{job_no}}`}}, {{`{{estimate_no}}`}}, {{`{{invoice_no}}`}}
  • {{`{{total}}`}}, {{`{{subtotal}}`}}, {{`{{tax}}`}}
  • {{`{{org_name}}`}}, {{`{{org_phone}}`}}, {{`{{org_email}}`}}
  • {{`{{line_items}}`}} - Auto-formats line items table

Using Templates

Mobile App

Progressive Web App (PWA)

Creative Job Hub works as a full mobile app without app store downloads.

Installation

iOS (Safari):

  1. Visit creativejobhub.com in Safari
  2. Tap the Share button
  3. Select "Add to Home Screen"
  4. Tap "Add"

Android (Chrome):

  1. Visit creativejobhub.com in Chrome
  2. Tap the menu (three dots)
  3. Select "Install App" or "Add to Home Screen"
  4. Tap "Install"

Desktop: Look for install icon in browser address bar or header.

Best Practices & Tips

Daily Workflow for Owners/Managers

  1. Morning:
    • Review dashboard for overnight activity
    • Check who's clocked in
    • Use "Plan My Day" for route optimization
    • Confirm today's schedule with technicians
  2. Throughout Day:
    • Monitor real-time job progress on map
    • Respond to customer messages promptly
    • Follow up on estimates sent
    • Review and approve time exceptions
  3. End of Day:
    • Convert completed jobs to invoices
    • Send invoices to customers

Troubleshooting

Login & Account Issues

  • Forgot Password: Use "Forgot Password" link, check email (including spam)
  • Can't Login: Try incognito/private mode, clear browser cache
  • Email Not Received: Check spam folder, verify email address, try resend
  • Two-Factor Issues: Contact support to reset if locked out

Mobile App Issues

  • App Won't Install: Try different browser (Safari on iOS, Chrome on Android)
  • Offline Mode Not Working: Check browser permissions, ensure enough storage space
  • GPS Not Working: Enable location services in device settings
  • Photos Won't Upload: Check camera permissions, verify internet connection
  • Clock In/Out Missing: Refresh app, check if you're in correct location
  • Push Notifications: Enable notifications in browser/device settings

Communication Issues

  • Emails Not Sending: Verify Resend integration, check spam reputation
  • SMS Not Sending: Verify Twilio credentials, check phone number format
  • Customer Not Receiving: Confirm correct email/phone, check their spam
  • Email Opens Not Tracking: Some email clients block tracking pixels

Payment & Billing Issues

  • Stripe Not Connected: Re-authorize in Integrations → Stripe
  • Payment Failed: Check Stripe dashboard for error details
  • Invoice Not Paid: Verify payment link works, check Stripe status
  • Subscription Issues: Go to Settings → Billing, update payment method

Integration Issues

  • QuickBooks Not Syncing:
    • Check connection status in Integrations
    • Reconnect if token expired (happens every 6 months)
    • Review sync logs for specific errors
    • Verify matching customer names
  • Twilio Errors:
    • Verify Account SID and Auth Token
    • Check Twilio balance
    • Confirm phone number active
    • Review Twilio error logs
  • Google Drive Not Syncing:
    • Reconnect Google account
    • Check folder permissions
    • Verify storage space available

Scheduling & Routing Issues

  • Plan My Day Not Working: Ensure jobs have valid addresses, check internet connection
  • GPS Tracking Inaccurate: Enable high-accuracy mode on mobile device
  • Route Not Optimizing: Verify Mapbox integration, check job locations

Time Tracking Issues

  • Missing Punches: Edit timecard in Time Admin to add missing entries
  • Wrong Location Recorded: GPS can be off by 10-50 feet; review in context
  • Overlapping Shifts: Check for duplicate clock-ins, resolve in Time Admin
  • Payroll Export Errors: Verify pay period dates, check for exceptions

Data & Reports Issues

  • Reports Not Loading: Try smaller date range, check internet connection
  • Missing Data: Verify data was entered, check filters applied
  • Export Failed: Reduce dataset size, try different format (CSV vs PDF)
  • Dashboard Empty: Requires at least some job/client data to display

Performance Issues

  • Slow Loading:
    • Clear browser cache and cookies
    • Close unused browser tabs
    • Check internet speed
    • Try different browser
